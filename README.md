# How to install project on local repo
- **Be sure that u have installed:**
> XAMPP
> Composer
> Source code editor
- **To install do:**
> Open the 'start.bat' file for Windows, or 'start.sh' for Linux
> Open terminal and type following command:
> > npm run dev
