## About Project

Project allows users to create and bid auctions of cars. Registration, login, administrator panel, source view and more.

### Project is using

- **Broadcast**
- **Routes**
- **Database**
- **Connection with database**
- **Views**
- **Schedules**
- **Seeders**
- **Models**
- **Bootstrap**
